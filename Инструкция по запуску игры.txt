Для запуска проекта в PyCharm следуйте следующим шагам:

Откройте PyCharm и выберите проект An analogue of the tower blocks game by Daniyar Ismailov . 

Убедитесь, что у вас установлены все необходимые зависимости для проекта. Чтобы установить зависимости, откройте терминал внутри PyCharm (внизу окна) и выполните команду pip install -r requirements.txt или pip install <имя зависимости> для каждой необходимой зависимости.

Откройте файл main.py , содержащий точку входа в проект.

Щелкните правой кнопкой мыши внутри файла и выберите опцию "Run" (Запустить) или нажмите сочетание клавиш Ctrl+Shift+F10 для запуска файла. 

PyCharm выполнит запуск проекта и вы увидите вывод в окне "Run" (Запуск) и "Terminal" (Терминал) внизу окна. В "Terminal" (Терминал) выберите уровень сложности для запуска игры (легкий / средний / сложный).

При некорректном выборе уровня сложности будет установлен уровень по умолчанию (средний).

Для запуска игры в окне нажмите клавишу "space" или "пробел"

Используйте клавиши для перемещения: "arrow left" или "стрела влево" для перемещения влево, "arrow right" или "стрела вправо" .
